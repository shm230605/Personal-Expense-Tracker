## 💸 Personal Expense Tracker

A modern **Finance & Expense Analytics Dashboard** built using **Python, Pandas, Matplotlib, and Streamlit**.  
It helps users track income, expenses, savings, and visualize spending patterns through interactive charts.

---

## 📌 Project Overview

This project is designed to simulate a real-world **SSpersonal finance management system**, similar to fintech apps like Google Pay insights or banking dashboards.

It provides:

- Income vs Expense tracking  
- Category-wise spending analysis  
- Payment method insights  
- Monthly and daily trends  
- Clean and interactive UI using Streamlit  

---

## 🚀 Features

### 💰 Finance Tracking
- Total Income calculation  
- Total Expenses calculation  
- Savings auto-calculation  

---

### 📊 Data Visualization
- Category-wise expense pie chart  
- Monthly spending trend  
- Daily expense analysis  
- Payment method breakdown  
- Summary dashboard  

---

### 🎛️ Interactive Dashboard
- Sidebar category filter  
- Dynamic data updates  
- Clean metric cards (Income, Spend, Savings)  

---

### 🧠 Smart Data Handling
- Auto-fallback dataset if CSV missing  
- Handles missing columns safely  
- Cleans invalid numeric data
