💸 Personal Expense Tracker

A modern Finance & Expense Analytics Dashboard built using Python, Pandas, Matplotlib, and Streamlit.
It helps users track income, expenses, savings, and visualize spending patterns through interactive charts.

📌 Project Overview

This project is designed to simulate a real-world personal finance management system, similar to fintech apps like Google Pay insights or banking dashboards.

It provides:

Income vs Expense tracking
Category-wise spending analysis
Payment method insights
Monthly and daily trends
Clean and interactive UI using Streamlit
